# NLP Final - 醫病訊息決策與對話語料分析競賽 NTUNLP_隊名想了十秒鐘

## Installation
### Requirements
* python==3.8
* torch==1.8.1+cu111
* pytorch-lightning==1.3.5
* pandas==1.2.4
* transformers==4.6.1
* datasets==1.8.0
* spacy==3.0.6
* spacy-legacy==3.0.5
* zh-core-web-md (spacy)
* tqdm

### Installing from the source
```shell
pip install -r requirements.txt
```
